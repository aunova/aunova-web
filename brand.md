# Aunova Brand Guidelines

## Typography

### Fonts
- **Headlines**: Source Serif 4 (variable font)
- **Body/UI**: Inter (variable font)
- **Monospace**: SF Mono, Monaco, Fira Code

### Type Scale
| Token | Size | Usage |
|-------|------|-------|
| `--text-hero` | 40–64px | Hero headlines |
| `--text-h1` | 30–44px | Page titles |
| `--text-h2` | 22–30px | Section titles |
| `--text-h3` | 18–20px | Subsections |
| `--text-body` | 17px | Body text |
| `--text-body-sm` | 15px | Secondary text |
| `--text-ui` | 14px | UI elements |
| `--text-xs` | 13px | Captions, labels |

## Design Principles

1. **Calm, Institutional** - Avoid playful or SaaS aesthetics
2. **Burgundy as Seal** - Use sparingly, like a stamp or accent
3. **Subtle Shadows** - Minimal, avoid "card floating" look
4. **Intentional Transitions** - Slow, deliberate (250ms+)
5. **Serif Headlines** - Source Serif 4 for gravitas
6. **Sans Body** - Inter for readability
